同济大学2020届本科生毕业设计

本项目是在同济大学区块链创新应用实验室宋志光教授的指导下完成的2020届本科生毕业设计，论文题目为：《疫情区块链微博系统设计与实现》。旨在掌握区块链的基本原理，实现区块链在软件领域的应用，并总结本科阶段学习的知识与技能，为四年的大学生活交出一份完整的答卷。

开源的微博社区

为了更快捷地完成设计目标，本项目基于Mastodon开源工程自主构建了一个具有完整功能的类微博社交网络，以便后续工作中有针对性地进行二次开发。这一工作流程让开发者体会到了开源精神的美好与重要性。

具有区块链特性的数据

利用NEM区块链，本项目可以通过转账的方式将相关信息附加在交易过程中。此时，区块的生成机制能够实现该信息在区块链上的存储，这也意味着最初被附加的消息拥有了区块链的特性，即去中心化、不可伪造、全程留痕、可以追溯、公开透明、集体维护等等。

为公共事业提供创新性支持

本项目整合了类微博社交网络和区块链两种计算机信息技术，创新性地提出了一种互联网应用的模型，即：将所有用户在公共平台的发言基于区块链进行存储。正如论文题目所言，这种存储方式的新特性，将会为诸如疫情防控等公共事业提供最有力的数据支持！

项目简介

本项目利用Mastodon开源工程搭建微博社区，通过数据库技术将用户发布内容上传至NEM区块链，赋予用户数据各种区块链特性。

此后，根据区块链的种种数据存储优势，本项目可为相关公共安全事业提供创新型的技术保障。

区块链（Blockchain）技术

区块链起源于比特币（Bitcoin）：2008年11月1日，一位自称中本聪（SatoshiNakamoto）的人发表了《比特币：一种点对点的电子现金系统》一文，阐述了一种基于加密技术、P2P网络技术、时间戳技术、区块链技术等的电子现金系统的构架理念，标志着比特币的诞生。两个月后，理论步入实践。2009年1月3日，第一个序号为0的创世区块诞生。几天后，2009 年1月9日，序号为1的区块出现，并通过特定密码学方法关联，与序号为0的创世区块相连接形成了链，标志着区块链的诞生。其中，每一个数据块包含了一批次比特币网络交易的信息，用于验证区块链信息的有效性（防伪）和生成下一个区块。

近些年来，以“比特币”为代表，一大批数字虚拟货币在市场上的交易愈发火爆。而伴随着这些货币兑美元价格在网络上的涨落波动，作为其背后的支撑技术，“区块链”这一概念也在信息技术领域越来越受到重视，甚至在全社会都获得了一定程度的关注。然而，由于理念的超前性和技术本身的复杂性，“区块链”一词并不具有一个被广泛接受的定义。从本质上讲，它是一个分布式的共享账本和数据库，存储于其中的数据或信息，具有去中心化、不可伪造、全程留痕、可以追溯、公开透明、集体维护等等特征。基于这些特征，区块链技术奠定了坚实的“信任”基础，创造了可靠的“合作”机制，保证了自身的“诚实”与“透明”。在基于区块链的应用场景中，这项技术能够非常容易地解决信息不对称问题，实现多个主体之间的协作信任与一致行动，具有相当良好的发展前景。

相关技术

为了实现毕业设计的既定需求，本项目的工作过程使用了多种计算机技术，代表了开发者对于相关知识的储备与学习水平。

Linux 操作系统

Linux，全称GNU/Linux，是一套免费使用和自由传播的类UNIX操作系统，也是一个基于POSIX和Unix的多用户、多任务、支持多线程和多CPU的操作系统。它支持32位和64位硬件。Linux性能稳定，有上百种不同的发行版，如基于社区开发的debian和基于商业开发的Red Hat Enterprise Linux等。本项目的服务器操作系统为：Ubuntu 18.04.4 LTS。

PostgreSQL 数据库

PostgreSQL是一个功能强大、特性齐全、源代码开放的客户端/服务器对象—关系型数据库管理系统。它支持大部分SQL标准并且提供了很多其他现代特性，其中许多领先概念在比较迟的时候才出现在商业网站数据库中。同时，因为许可证灵活，任何人都可以以任何目的免费使用、修改和分发PostgreSQL。本项目使用的数据库版本为：PostgreSQL 10。

MySQL 数据库

MySQL是一种已开放源代码的关系型数据库管理系统（RDBMS），使用最常用的数据库管理语言——结构化查询语言（SQL）进行管理。由于其开源特性，任何人都可以在General Public License的许可下下载并修改，同时在速度、可靠性和适应性等方面颇具竞争力。在无需事务化处理时，MySQL几乎是管理内容最好的选择。本项目使用的数据库版本为：MySQL 5。

Java 计算机编程语言

Java是一门面向对象编程语言，不仅吸收了C++语言的各种优点，还摒弃了C++里难以理解的多继承、指针等概念，具有简单易用、面向对象、分布式、健壮性、安全性、平台独立与可移植性、多线程、动态性等特征。作为静态面向对象编程语言的代表，Java极好地实现了面向对象理论，允许程序员以优雅的思维方式进行复杂的编程。Java功能强大，可以用于编写桌面应用程序、Web应用程序、分布式系统和嵌入式系统应用程序等。

Python 计算机编程语言

Python是一种跨平台的计算机程序设计语言，也是一个高层次的结合了解释性、编译性、互动性和面向对象特性的脚本语言。Python最初被设计用于编写自动化脚本（shell），随着版本的不断更新和语言新功能的添加，越多被用于独立的、大型项目的开发。

Github 代码仓库

GitHub于2008年4月10日正式上线，是一个面向开源及私有软件项目的托管平台，只支持Git作为唯一版本库格式进行托管。除了Git代码仓库托管和基本Web管理界面外，GitHub还提供了订阅、讨论组、在线文件编辑器、文本渲染、协作图谱（报表）、代码片段分享（Gist）等功能。目前，Github的注册用户已经超过了350万，托管版本数量也是非常之多，其中不乏知名开源项目，如：Ruby on Rails、jQuery等。
